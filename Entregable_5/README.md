# Entregable 5 - Experimentación y métricas de red

Este entregable presenta la fase de experimentación cuantitativa del proyecto, enfocada en la medición real del desempeño de la red implementada en GNS3. Se evaluaron métricas como latencia, jitter, throughput, retransmisiones TCP, pérdida de paquetes, comportamiento de STP ante falla de enlace y congestión mediante tráfico background.

## Contenido

- Metodología experimental.
- Prueba de latencia y jitter mediante ping extendido.
- Prueba de throughput mediante transferencia TCP.
- Prueba de fallo de enlace con STP.
- Prueba de congestión con tráfico background.
- Tabla resumen de experimentos.
- Comparación general de resultados.
- Capturas de pantalla de cada experimento.
- Análisis de métricas obtenidas.
- Conclusiones del entregable.
