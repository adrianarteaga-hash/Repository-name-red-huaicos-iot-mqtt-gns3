# Entregable 1 - Definición del problema y estado del arte inicial

Este entregable presenta el planteamiento inicial del proyecto, incluyendo el tema de investigación, problema, preguntas de investigación, objetivos, justificación, estado del arte preliminar, tecnología emergente propuesta, esquema lógico y métricas iniciales estimadas.

## Contenido

- Tema de investigación.
- Problema de investigación.
- Preguntas de investigación.
- Objetivo general y objetivos específicos.
- Estado del arte inicial.
- Justificación.
- Tecnología emergente propuesta.
- Esquema lógico.
- Métricas iniciales propuestas.
