# Entregable 4 - Integración de tecnología emergente y análisis de protocolos

Este entregable presenta la integración de la tecnología emergente IoT/MQTT dentro de la red diseñada para el monitoreo y alerta temprana de huaicos. Se explica cómo el sensor IoT transmite telemetría hacia un broker Mosquitto ubicado en la red central, y se analiza cómo MQTT interactúa con protocolos clásicos del curso como ARP, ICMP y TCP.

## Contenido

- Introducción a la integración de tecnología emergente.
- Descripción de la integración IoT/MQTT en la topología.
- Diagrama de flujo de datos con MQTT.
- Análisis de protocolos afectados.
- Capturas de Wireshark de tráfico ARP, ICMP, TCP y MQTT.
- Recepción de telemetría MQTT en el broker Mosquitto.
- Comparación de overhead entre tráfico normal y MQTT.
- Métricas específicas de la integración MQTT.
- Conclusiones del entregable.
