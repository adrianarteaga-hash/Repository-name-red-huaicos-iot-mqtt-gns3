# Entregable 3 - Diseño de red y simulación base

Este entregable presenta la primera versión funcional de la red diseñada para el monitoreo y alerta temprana de huaicos. Se desarrolla la topología base en GNS3, incluyendo segmentación mediante VLAN, direccionamiento IPv4, enrutamiento entre redes, STP activo y pruebas iniciales de conectividad.

## Contenido

- Diseño detallado de la topología de red.
- Tabla de VLANs implementadas.
- Tabla de direccionamiento IPv4.
- Configuración de STP entre switches.
- Simulación base funcional en GNS3.
- Verificación de VLANs.
- Prueba de conectividad entre redes.
- Captura de tráfico en Wireshark.
- Evidencia de trama con etiqueta 802.1Q.
- Primeras métricas de ping, STP y throughput básico.
